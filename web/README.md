# One Piece Frontend

This Vite + React app renders the notebook graphs and interactive controls.

## Setup
1. Generate data:

   python scripts/build_web_data.py

2. Install dependencies:

   cd web
   npm install

3. Run locally:

   npm run dev

## Deploy (Vercel)
- Framework: Vite
- Root directory: web
- Build command: npm run build
- Output directory: dist
